# moka

## Purpose
This is an application that provides management for a real horeca machinery warehouse in some company.
This backend handles the database that contains all the sotre information: Machines, stores, inputs, ouputs and so on.
These machines are provided to the clients of the company under a contract and usually the machine comes back when the contract is finished.
All these machines have a value, and is very important to have all the machine stock under control, and have every machine tracked:
- Which machines are avalaible in our store.
- Which machines are intalled in customers, and when were installed.
- To know the machine state: New, used, broken, repaird and so on.

## Technologies
- ASP.NET Core 8
- Entity Framework Core
- SQL Server
- Swagger/OpenAPI
